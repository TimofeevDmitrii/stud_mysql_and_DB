5. Количество сотрудников определенного возраста

Инструкция по использованию платформы



Имеется таблица (сущность) с персоналом staff.
У сущности имеются следующие поля(атрибуты):
id – идентификатор;
firstname – имя;
lastname - фамилия
post - должность,
seniority – стаж;
salary – заработная плата;
age - возраст.

Посчитайте и выведите количество сотрудников с должностью 'Рабочий' и возрастом не моложе 24 лет и не старше 49 лет.


-- Мое решение:

----- Код --------
-- Вы работаете с MySQL
-- Введите свой код ниже

SELECT COUNT(post) as 'Количество работников с должностью "Рабочий" от 24 до 49 лет' FROM staff
WHERE post='Рабочий' AND (age >= 24 AND age <= 49);

----- Код --------


Идеальное решение в автотестах:

----- Код --------
-- Вы работаете с MySQL
-- Введите свой код ниже

SELECT COUNT(*) AS "количество"
FROM staff
WHERE post = 'Рабочий' AND age BETWEEN 24 AND 49;

7. Сотрудники младше 30 лет

Инструкция по использованию платформы



Имеется таблица (сущность) с персоналом staff.
У сущности имеются следующие поля(атрибуты):
id – идентификатор;
firstname – имя;
lastname - фамилия
post - должность,
seniority – стаж;
salary – заработная плата;
age - возраст.

Найдите средний возраст сотрудников по каждой должности из сущности staff.
Выведите только те должности, у которых средний возраст меньше 30 лет.


-- Мое решение

----- Код --------
-- Вы работаете с MySQL
-- Введите свой код ниже

SELECT post FROM 
(SELECT post, AVG(age) AS worker_avg_age FROM staff
GROUP BY post
HAVING worker_avg_age < 30) as avg_age_posts;


----- Код --------

-- Идеальное решение



----- Код --------
-- Вы работаете с MySQL
-- Введите свой код ниже

FROM staff
GROUP BY post
HAVING AVG(age) < 30;
